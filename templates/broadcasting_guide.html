Broadcasting Guide
Broadcasting Guide (Read This First)

This service works best when these rules are followed.

  • Wi-Fi is NOT allowed. Broadcasting must be done using a wired Ethernet connection.
  • A mixer is strongly recommended. Broadcasting without one will result in poor audio quality.

This guide is intentionally simple. Follow it exactly.

Critical Requirements
  • Wired Internet (Ethernet Only)
    • Do NOT broadcast over Wi-Fi.
    • Wi-Fi causes dropouts, lag, and unstable audio.
    • If you are on Wi-Fi, stop and plug in an Ethernet cable.
  • Audio Mixer (Highly Recommended)
    • All microphones are supported.
    • Using a mixer provides stable, clean, and consistent audio.
    • Without a mixer, audio is often quiet, distorted, or unbalanced.
    • Poor audio quality leads to listener complaints.
Rocket Broadcast
Recommended Equipment
  • Microphone (any type supported)
  • Audio Mixer (USB or analog)
  • Ethernet Cable (connected directly to router or switch)
  • Stable Internet (minimum 1 Mbps upload)
How to Broadcast
1. Open the Broadcasting Page
  • Use a computer, tablet, or phone.
  • Confirm Ethernet is plugged in.
  • Allow microphone access in your browser.
2. Enter the Required Information
  • Colony Name
  • Description
  • Password
    • Same password for all authorized broadcasters.
    • Contact the HBNI Audio admin if you don’t have it.
Testing (Required)
  • Set Colony Name to test.
  • Speak for 10–20 seconds.
  • Confirm:
    • Audio is clear
    • No cutting out
    • Ethernet is connected
  • If it fails:
    • Adjust microphone or mixer levels
    • Check microphone connection
    • Confirm Wi-Fi is disabled
Go Live
  • Enter your real Colony Name.
  • Click Start Broadcasting.
  • Speak clearly.
  • Do not unplug Ethernet.
  • Do not change networks.
Public vs Private Broadcasts
Public
  • Visible on listeners page.
  • Automatically archived.
  • You receive a copy.
Private
  • Hidden from listeners page.
  • Not publicly archived.
  • You still receive a copy.

Common Problems
  • Audio cutting out → Network issue, usually Wi-Fi.
  • Quiet or distorted audio → No mixer or poor mic setup.
  • Listener complaints → Audio quality too low.

Fix: Use Ethernet. Use a mixer.

Final Reminder

Ethernet is required. A mixer prevents bad audio.
